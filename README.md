# httpServer
a base http server applications use nodeJS 
You can upload an img and see the img on the server
construction
```
.
├── index                // 主文件
├── server               // 服务器的代码
├── router               // 路由
├── requestHandlers      // 对于每一个路由的请求处理程序
.
```
please exec the command to run the server:
```
node index.js
```
